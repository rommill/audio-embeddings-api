# Audio Embeddings API

An AI-powered service that generates vector embeddings for audio files using Google's YAMNet and allows for similarity searches.

## Tech Stack

- **Language:** Python 3.12
- **Framework:** FastAPI
- **AI Model:** YAMNet (TensorFlow)
- **Database:** SQLite

## Setup Instructions

1. **Create Virtual Environment:**
   ```bash
   python -m venv venv
   source venv/bin/activate
   ```

## Setup & Running

### Option 1: Using Docker (Recommended)

```bash
docker compose up --build
```
